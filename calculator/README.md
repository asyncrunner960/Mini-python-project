# Console Calculator

A simple command-line calculator built with Python as one of my first projects.
It performs basic arithmetic operations and was my introduction to
user input, conditional logic, and error handling.

## Features
- Addition, subtraction, multiplication, and division
- Floor division and modulo operations
- Handles division by zero with a clear error message
- Input validation prevents crashes on invalid data

## How to Run
1. Make sure you have Python 3 installed
2. Download `calculator.py`
3. Open terminal and run:
```
python calculator.py
```
4. Enter two numbers and an operator when prompted

## Example
```
Enter first number: 10
Enter second number: 5
Enter operation: +
Result: 15.0
```

## What I Learned
- Getting user input with `input()`
- Converting types with `int()` and `float()`
- Conditional statements (`if` / `elif` / `else`)
- Basic error handling
